Cadastro de Usuários - CRUD simples

O Projeto

O projeto consiste em um CRUD completo para gerenciamento de usuários. Implementação de uma API RESTful e uma interface reativa em Vanilla JS (Vite), demonstrando as operações essenciais de criação, leitura, atualização e exclusão de dados.
